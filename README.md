Install
-------
You need cabal-dev.
 1. $ git clone https://github.com/mathfur/grep-tree.git
 2. $ cd grep-tree
 3. $ cabal-dev install --only-dependencies
 4. $ cabal-dev configure
 5. $ cabal-dev build
 6. Add dist/build/grep-tree/grep-tree to PATH
